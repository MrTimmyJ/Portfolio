Project Overview

        Cubicka is a fully custom 3D voxel engine built with LWJGL and OpenGL, focused on shader-driven graphics, intuitive UI/UX, and efficient terrain rendering. Inspired by games like Minecraft or Cube World, but tailored for performance, customization, and creative expansion.

        Cubicka is more than just blocks—it's a voxel engine designed for adventure and exploration. Featuring programmable shaders, structured terrain generation, modular chunking, and entity systems, Cubicka aims to serve as both a sandbox and a foundation for a voxel-based experiences.

Project Features

  • 🌍 Chunk-Based Voxel Terrain – Procedural terrain using noise functions and customizable biomes.
  • 🧱 3D Block World – Dynamic block ID array generation per chunk with VBO/VAO GPU rendering.
  • 💡 Lighting & Shaders – Custom GLSL shaders and lighting with directional lights.
  • 🧭 Biomes & Structures – Biome-specific structures and entities.
  • 🔁 Chunk Culling – Only visible chunks are rendered using frustum culling for performance.
  • 👁️ Camera & Controls – Entity-driven camera and player movement.
  • 🧪 Entity System – Flexible system for adding trees, mobs, structures, etc.
  • 🖱️ UI/UX First – Clean interaction systems and user friendly menus.

Gameplay

You move your character around to explore a randomly generated world.

Controls:

  • ↑ / W..............Move Forward
  • ↓ / S..........Move Backward
  • ← / A...........Move Left
  • → / D.........Move Right
  • Space.........Jump

Technologies Used

  • ☕ Java 14
  • 🧱 LWJGL 3 (OpenGL bindings)
  • 📐 JOML (math library)
  • 🎨 GLSL (custom shaders)
  • 🔂 Buffered VBO/VAO rendering
  • 🧠 Perlin/Simplex Noise for procedural terrain

Overview

Confract solves a real problem in software teams: documentation sprawl. When multiple services, APIs, or teams each maintain their own docs, the result is contradictory, duplicated, or fragmented information that's hard to navigate.

Confract takes multiple documentation sources as input and produces a single merged, deduplicated, and restructured output. It uses NLP to understand semantic relationships between sections — so it doesn't just concatenate files, it understands which sections are about the same topic and merges them intelligently.

The tool is exposed as both a command-line interface and a REST API, making it easy to integrate into documentation pipelines or use ad hoc.

Workflow

Provide source documents
Pass multiple documentation files as input — Markdown, text, or HTML.
Parse and embed
Each document is parsed into sections. Sections are semantically embedded for similarity comparison.
Match and merge
Semantically similar sections are identified and merged. Unique sections are preserved as-is.
Reconstruct output
Merged sections are reassembled into a coherent document with logical heading hierarchy.
Export
Output written as Markdown or returned as JSON via the API. Ready for use in docs pipelines.

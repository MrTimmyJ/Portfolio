Overview

Next Novel is a book recommendation API that goes beyond genre tags. Instead of simple category matching, it uses semantic analysis to understand the thematic content of books and find genuine similarity — matching mood, themes, and writing style rather than just shelf labels.

Users submit a list of books they've enjoyed and receive ranked recommendations with explanations. The API is stateless and designed to be embedded in any reading application or used as a standalone service.

The project also served as a proving ground for automated CI/CD with GitHub Actions — every push triggers tests and a deployment pipeline, making releases fully automated.

Architecture

🔍
Semantic Engine
  • Text embedding for book descriptions
  • Cosine similarity ranking
  • Multi-factor preference weighting
  • Thematic cluster detection
🚀
REST API
  • POST /recommend with reading history
  • JSON response with ranked results
  • Input validation and error handling
  • Stateless, horizontally scalable
🔄
CI/CD
  • GitHub Actions workflow on push
  • Automated test suite
  • Deployment pipeline on merge to main
  • Failure notifications

Features

Workflow

Submit reading history
POST request with a list of books the user has read and enjoyed.
Embed and analyze
Each book is embedded as a semantic vector. User preferences are modeled from the input list.
Rank candidates
Cosine similarity computed against the book catalog. Results ranked by score.
Return recommendations
Top N results returned as JSON with titles, scores, and thematic reasoning.

Overview

Dev Tools Suite is a set of browser-based utilities for everyday developer tasks. The philosophy: fast, zero-install, no account required. Open the URL, use the tool, done.

Everything runs client-side in vanilla JavaScript — no backend, no frameworks, no build step. This keeps the tools instant to load, auditable, and hostable anywhere. It also made it a good exercise in writing clean, maintainable JS without relying on library abstractions.

Tools Included

📋
JSON Formatter
  • Pretty-print and minify JSON
  • Syntax validation with error highlighting
  • Copy to clipboard
🔍
Regex Tester
  • Live match highlighting
  • Flag toggles (g, i, m, s)
  • Match group extraction
🔐
Base64 Encoder
  • Encode and decode text
  • URL-safe variant support
🎨
Color Converter
  • HEX ↔ RGB ↔ HSL conversion
  • Color picker input
  • Visual preview
🕐
Timestamp Tools
  • Unix ↔ human-readable conversion
  • Current timestamp
  • Timezone aware
🔤
String Utilities
  • Case conversion (camel, snake, kebab)
  • Word / char count
  • URL encode/decode

Design Decisions

⚡
Zero dependencies
No npm, no bundler, no framework. Vanilla JS and CSS only — fast load, easy to audit, works offline.
🔒
Client-side only
No data is sent to a server. Everything runs in the browser — safe to use with sensitive content.
📱
Responsive layout
Tools work on mobile and desktop. Useful for quick checks on the go.

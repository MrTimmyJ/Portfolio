Project Details

        A full-stack produce e-commerce site developed as a 2024 group project by a team of 7 students in Evergreen‚Äôs Data Structures & Algorithms course. Built with a Go-based API, a dynamic frontend UI, and a Bootstrap-powered admin dashboard, the app supports browsing, searching, purchasing, and managing produce.


TESCSE-Ecom simulates a small online produce store, allowing users to:

- Browse seasonal fruits and vegetables

- Search and filter items

- Create user accounts and place orders

- View shopping carts

- Stay up to date with local fresh produce and community events

It served as a capstone for group collaboration, sprint development, and full-stack architecture using Go and modern web technologies.
